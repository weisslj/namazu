template
--------

NMZ.head          ... Header of search results.
NMZ.foot          ... Footer of search results.
NMZ.body          ... Query description. Displayed when no keyword is given. 
NMZ.tips          ... Tips for searching.
                      Displayed when no document is retrieved. 
NMZ.result.normal ... File to specify the style of search results. (normal)
NMZ.result.short  ... File to specify the style of search results. (short)


language
--------

NMZ.*             ... basic template. (US-ASCII)
NMZ.*.ja          ... japanese (UTF-8)
NMZ.*.en          ... english (UTF-8)
NMZ.*.fr          ... french (UTF-8)
NMZ.*.de          ... deutsch (UTF-8)
NMZ.*.es          ... spanish (UTF-8)
NMZ.*.pl          ... polish (UTF-8)
