Namazu メイリングリスト

Last Modified: 2/15/1999

概要

全文検索システム Namazu について議論するメイリングリストです。 Namazu に興味のある方はぜひ参加してください。

Namazu メイリングリスト は 1/24/1998 に開設されました。2/2/1999 現在、 238 のメイルアドレスが登録され、合計 1960通のメイルが流通し ています。

何か不明な点があれば satoru@isoternet.org までご連絡ください。

おしらせ

2/2/1999 からメイリングリストのホストが ring.gr.jp に移動しました。 Ring Server Project のお世話に なっています。古いアドレスに投稿された 1960通の記事を アーカイヴにまとめました。

参加方法

メイリングリストへの参加(登録)は 本文に subscribe namazu とだけ書 いたメイルを Majordomo@ring.gr.jp へ送ってください。登録は手動となっ ています。

投稿先は namazu @ ring.gr.jpです。投稿されたメイルは Webなどで公開されること があります。

退会方法

本文に unsubscribe namazu とだけ書いたメイルを Majordomo@ring.gr.jp に送ります。その際、 From: はメイリングリスト に登録しているアドレスに設定してください。

メイルアドレスの変更方法

新しいメイルアドレスを登録してから古いアドレスを削除してください。

お願い

メイリングリストに質問をする前に必ず manual.html, FAQ をじっくり読んでください。また、UNIX の 使い方など、 Namazu以前の初歩的な質問はご遠慮ください。
